# Voise

**Анализатор вокала в реальном времени**

Приложение для анализа вокала с определением:
- ✅ **Попадания в ноту** с точностью до центов
- ✨ **Чистоты звука** (HNR, спектральный анализ)
- 🎤 **Типа голоса** (фальцет/грудной/микст)

---

## Что нового в этой версии (v2.0)

### 🚀 Новый функционал
- **Piano Roll** — визуальное отображение нот на временной шкале. Позволяет видеть, как меняется высота твоего голоса во времени, и сравнивать с эталонной мелодией.
- **Mixt Analyzer** — анализ переходного регистра (микста). Помогает определить, где твой голос переключается между грудным и головным резонаторами, и тренировать плавность переходов.

### 🔧 Исправления
- **Исправлен главный источник лагов записи**: раньше очередь аудио-чанков росла бесконечно, если анализ не успевал за микрофоном — задержка накапливалась и всё "тормозило" сильнее чем дольше шла запись. Теперь буфер всегда обрезается до самых свежих данных.
- **Исправлен лаг MIDI-клавиатуры**: раньше каждый клик по клавише заново открывал/закрывал аудио-поток устройства (десятки-сотни мс блокировки прямо в обработчике клика) плюс блокирующий `time.sleep()` при отпускании. Теперь поток открывается один раз при запуске и живёт всё время работы — смена ноты происходит мгновенно.
- **Исправлена "октавная ошибка" при определении ноты**: раньше алгоритм (сырой YIN) мог спутать частоту с её половиной — например, при пении высокой ноты головным регистром иногда засчитывалось "идеальное попадание" в ноту на октаву ниже. Теперь используется `librosa.pyin` (вероятностный YIN с трекингом по времени) плюс отдельная проверка по спектру, которая ловит именно такие октавные ошибки и исправляет их.
- **Устранена утечка эталонного тона в микрофон**: пока звучит референсная нота с клавиатуры (и ещё короткое время после), анализ голоса приостанавливается — иначе микрофон мог уловить звук из динамиков и засчитать его вместо реального пения.

### ⚡ Оптимизация
- Расширен диапазон MIDI-клавиатуры (5 октав вместо 3), чтобы можно было выбирать по-настоящему высокие ноты, соответствующие диапазону поиска pitch-детектора.
- **Анализ вынесен в отдельный поток** — тяжёлые вычисления (pitch-detection, фильтрация, LPC) больше не блокируют интерфейс. Приложению разрешено использовать ресурсы CPU без искусственных ограничений.
- ⚡ Полосовой фильтр Баттерворта теперь считается один раз, а не на каждом кадре — меньше нагрузка на CPU.
- ⚡ Вычисление формант (LPC) векторизовано через `scipy.linalg.solve_toeplitz` вместо ручных Python-циклов.

### 🛡️ Стабильность
- **Крах-логи**: при любом необработанном исключении (и даже при низкоуровневых крашах вроде segfault) в папке `logs/` создаётся понятный текстовый отчёт `crash_*.txt` с датой, версиями библиотек и полным traceback, плюс общий журнал `logs/voise.log`. Открыть папку с логами можно через меню **Помощь → Открыть папку с логами**.
- Приложение больше не падает целиком при отсутствии микрофона, повреждённом `config.json` или ошибке в отдельном кадре анализа — ошибки логируются, а не роняют программу.

### 🎨 Интерфейс
- Новый **индикатор уровня сигнала** (VU-метр) — видно, что микрофон вообще что-то улавливает.
- Современная светлая/тёмная тема (переключается в Настройках), статус-бар с временем записи и параметрами.
- ⚙️ Настраиваемая скорость обновления анализа (мс) — компромисс между отзывчивостью и нагрузкой на CPU.

## Основные функции

- **Виртуальная MIDI-клавиатура** для выбора целевой ноты (5 октав)
- **Piano Roll** — визуализация нот во времени
- **Mixt Analyzer** — анализ переходного регистра
- **Анализ в реальном времени** (обновление 10 раз/сек)
- **Интеллектуальная фильтрация шумов** (noise gate, bandpass filter)
- **Точное определение pitch** (YIN алгоритм, ±10 центов)
- **Визуализация результатов** с цветовой индикацией
- ⚙️ **Полностью настраиваемые параметры** анализа
- **100% локальная обработка** — никаких сетевых запросов

---

## Анализ вокала

- **Высота тона**: определение ноты и частоты
- **Попадание в ноту**: процент точности и отклонение в центах
- **Качество звука**: оценка чистоты (0-100) на основе HNR
- **Тип голоса**: грудной, фальцет или микст (анализ формант и гармоник)

## Быстрый старт

### Windows (рекомендуется)

1. **Скачайте проект** в `C:\Users\<YourUser>\Desktop\Voise`
2. **Запустите** `install.bat` (автоматическая установка)
3. **Запустите** `run.bat`
4. **Пойте!**

### Ручная установка

```bash
# Клонируйте репозиторий
cd C:\Users\<YourUser>\Desktop\Voise

# Создайте виртуальное окружение
python -m venv venv

# Активируйте
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/macOS

# Установите зависимости
pip install -r requirements.txt

# Запустите
python main.py
